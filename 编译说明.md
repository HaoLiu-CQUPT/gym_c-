# 编译运行说明

## 前提条件
必须先安装Qt库（即使不安装Qt Creator），可以从以下方式获取：
1. Qt官网下载Qt安装包（最小安装，只要Qt库和qmake）
2. 使用包管理器：vcpkg、conan等

## 命令行编译步骤（Windows）

### 1. 检查Qt是否已安装
在命令行输入：
```bash
qmake --version
```
如果显示版本号，说明已安装。

### 2. 生成Makefile
```bash
qmake StudentManagementSystem.pro
```

### 3. 编译项目
```bash
mingw32-make
```
或使用MSVC：
```bash
nmake
```

### 4. 运行程序
```bash
release\GymManagementSystem.exe
```
或
```bash
debug\GymManagementSystem.exe
```
